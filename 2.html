Decorders W3C Brazil
Ainda dá tempo para participar do Decoders W3C.br Você ainda pode participar do desafio de criar uma aplicação criativa utilizando as tecnologias da Open Web Plataform para uma nova visualização dos dados da localização dos escritórios do W3C no mundo. Utilize APIs de geolocalização, Canvas, SVG e outras tecnologias livres para criar novas soluções para dados de localização geográfica do W3C. Se a sua aplicação for escolhida, você pode apresentar seu projeto no Mozilla Hack Day no sábado, dia 1 de setembro. Maiores detalhes sobre o desafio, acesse http://decoders.w3c.br/

Vagas
UI HTML Developer
Esse profissional precisa respirar JSP/JSF, estar confortável para trabalhar com bibliotecas JSF como Primefaces, ICE e até mesmo o Websphere Commerce Server (embora entenda que o perfil WCS é quase impossível buscar fora da nossa base). Além desse conhecimento, javascript é mandatório, com plus para Dojo caso o candidato tenha experiencia. Esse profissional vai consumir diariamente webservices (e criar em alguns momentos também). Não apenas JSON como formatos abertos estao no escopo. Não houve definição de arquitetura ainda em relacao a Microdata, mas será um plus tambem. O perfil provavelmente será entre pleno e sênior, de acordo com a experiência do candidato aprovado. A URL dessa vaga no site da IBM é: https://jobs3.netmedia1.com/cp/job_summary.jsp?job_id=GBS-0507886

Webdesigner
Esse profissional é mais júnior/pleno, mas precisa pelo menos dominar markup e conseguir montar um foundation semantico, acessivel e validado, oriundo de wireframes. Ele precisa tambem ter conhecimento de prototipacao responsiva pq em alguns ciclos do sprint, ele poderá ser o responsável por montar não só o foundation, mas o wireframe tambem. Nesse contexto, HTML e CSS são mandatórios, com conhecimento em Dojo ou javascript um plus. Manipulação de suites graficas é necessário, mas esse profissional nao vai passar o dia fazendo recorte no photoshop (nao consigo - nem quero) A URL dessa vaga no site da IBM é: https://jobs3.netmedia1.com/cp/find.ibm.jobs/BR/Webdesigner/GBS-0507890/job/
