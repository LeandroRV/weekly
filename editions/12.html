Novidades
Ratchet
Confira o Ratchet, mais um projeto criado por Dave Gamache, Connor Sears e Jacob Thornton, ele consiste em uma biblioteca de componentes para criar protótipos para iPhone com HTML, CSS e JS, para quem ainda não deu uma olhada, clique e confira esse projeto.

Desafio CSS – Criar um Sol estiloso
Confira mais um desafio de CSS criado pelo Bernard de Luna. Esse desafio foi feito em 15 minutos e a idéia é criar um sol diferente, algo mas parecido com um “clock sun”. Ao final de cada desafio, será feito um sorteio entre os acertadores.
