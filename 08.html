Eventos
FrontInterior
Lançado o site do primeiro evento de front-end no interior paulista. O nome veio da intenção de criar um evento itinerante pelo interior paulista, onde a cada ano uma cidade sediará o evento. Este ano, ele ocorre no coração do Estado de SP, a cidade de Bauru, dia 24 de Novembro. A intenção é fazer com que o conhecimento se dissipe dos pólos metropolitanos e chegue com a mesma força em várias regiões. Além disso, traz possibilidade de maior integração entre a comunidade de desenvolvedores.

I Dojo Beaga JS
No próximo dia 15 vai rolar em conjunto com GDG (Google Developer Group)-BH um dojo para resolver um tema real com as soluções do Google Maps. Será explicado, de forma breve, como funciona a API de Google Maps e depois entre os participantes se tentará resolver um problema real, através de um Dojo. O evento é gratuito e é necessário fazer o cadastro, pois as vagas são limitadas.
 
Bibliotecas & Ferramentas
Destaque - Slideshow com efeito parallax
Direto do departamento open source da Globo.com, mais um plugin feito em jQuery para slideshow de imagens, só que dessa vez com efeito parallax, suporte para transições com CSS3 e gestos com swype.

Rangy - Seleção de texto
Uma biblioteca cross-browser para seleção de textos com diversas funcionalidades. Útil para delimitar highlights, seleções parciais em inputs, etc.

Monocle - Leitor e-books em JavaScript
O monocle é uma biblioteca que proporciona uma interação muito interessante para leitura de livros online/e-books.
